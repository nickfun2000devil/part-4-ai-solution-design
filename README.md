# Part 4: AI Solution Design — Agentic AI in Finance

## Overview
This project presents an AI solution design for applying Agentic AI in the Finance domain.
It covers problem definition, data requirements, model selection, evaluation, and responsible AI considerations.

## Domain
Finance

## Problem
Financial institutions rely on slow, manual processes for fraud detection, loan approvals,
portfolio management, and customer support. This project proposes an Agentic AI system
that can autonomously handle these tasks in real time.

- `solution_report.md` — Full solution report covering all 8 tasks
- `diagrams/solution_architecture.png` — System architecture diagram
- `ai_usecase_reference_catalog.csv` — Reference catalog used for domain selection
- `business_kpi_sample.csv` — KPI reference used for business impact analysis
- `data_dictionary__2_.md` — Data dictionary for reference files
